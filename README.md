# Vitality - Calculadora de Peso Ideal PWA 🚀✨

**Vitality** es una aplicación web progresiva diseñada para ayudarte a conocer tu rango de peso saludable basándose en fórmulas científicas y una estética premium.

## Características Principales 🌈

- **Cálculo Científico**: Utiliza las fórmulas de **Lorentz** y **Devine** para ofrecer un rango preciso. ⚖️
- **Experiencia Premium**: Diseño moderno con **Tailwind CSS**, **Glassmorphism** y animaciones suaves. 🎨
- **PWA (Progressive Web App)**:
    - Instalable en dispositivos móviles y escritorio. 📱
    - Soporte **offline** mediante Service Workers. 📡
    - Manifiesto configurado con iconos de alta resolución. ✨
- **Arquitectura Modular**: Código separado en HTML, CSS y JS para máxima escalabilidad y limpieza. 🧹

## Estructura del Proyecto 📂

- `index.html`: Página principal de captura de datos.
- `resultado.html`: Visualización dinámica de los resultados.
- `css/`: Estilos personalizados y sistema de diseño. 🎨
- `js/`: Lógica de cálculo y controladores de UI. 🧠
- `icons/`: Activos visuales para la PWA. 🖼️
- `sw.js`: Service Worker para el manejo de cache.
- `manifest.json`: Configuración de la aplicación móvil.

## Instalación y Uso 🛠️

1. Abre `index.html` en cualquier navegador moderno.
2. Introduce tu sexo, edad, altura y peso actual.
3. ¡Descubre tu peso ideal! 🌟
