# Estilos de la PWA 🎨

Este directorio contiene los estilos CSS personalizados que complementan a **Tailwind CSS**.

## Características 🌟
- **Tokens de Color**: Definidos en `:root` para asegurar consistencia.
- **Glassmorphism**: Efectos de desenfoque y transparencia para una sensación "premium".
- **Responsive Design**: Ajustes específicos para dispositivos móviles y aplicaciones instaladas (PWA).
- **Animaciones**: Transiciones suaves entre estados y páginas. ✨

## Uso 🛠️
Se vincula en el `<head>` de todos los archivos HTML:
```html
<link rel="stylesheet" href="css/styles.css">
```
